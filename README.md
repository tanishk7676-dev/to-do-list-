# 📝 To-Do List (Python)

This is a simple To-Do List application I built using Python while learning Object-Oriented Programming (OOP).

It allows users to add, remove, and view tasks in a clean and easy way. I also added a check to prevent duplicate tasks.

## 🚀 What it can do
- Add new tasks
- Remove existing tasks
- Show all tasks
- Avoid duplicate entries

## 🛠️ What I used
- Python
- OOP concepts (classes and methods)

## ▶️ How to run
1. Download the file
2. Open terminal in that folder
3. Run:
   python todo.py
